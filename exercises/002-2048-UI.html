2048 UI in React

Instructions

In this challenge, you will create the UI for the 2048 game, as it exists here, using React.

Start here.

Copy the files onto your local machine.

The index.html file includes everything you need to write JSX inside of the file. It loads React and Babel from a CDN.

Start by creating components for the Cell and Row, and then put them all together to display the complete game.

This is the same as yesterday, but now we are adding JSX instead of just plain JavaScript.

The functions you need to use from the provided JavaScript library, and the CSS that's available, the same as in the React without React exercise.
